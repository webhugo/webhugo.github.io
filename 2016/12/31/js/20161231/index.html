call apply bind

发表于 2016-12-31

call和apply

  1. call和apply都是Function.prototype的一个方法，是js引擎内在实现，两个方法作用一样，方式不一样而已(参数传递不一样)

  2. call 和 apply 都是为了改变某个函数运行时的上下文（context）而存在的，换句话说，就是为了改变函数体内部 this 的指向

  3. 从上面看，就是改变了this指针的调用者,a.setMessage.call(b,”haha”)相当于b.setMessage(“haha”),而apply传递的是数组

  4. 当一个 object 没有某个方法（本栗子中banana没有say方法），但是其他的有（本栗子中apple有say方法），我们可以借助call或apply用其它对象的方法来操作

    bind

  5. bind()方法会创建一个新函数，称为绑定函数，当调用这个绑定函数时，绑定函数会以创建它时传入 bind()方法的第一个参数作为 this，传入 bind() 方法的第二个以及以后的参数加上绑定函数运行时本身的参数按照顺序作为原函数的参数来调用原函数。
  1. 区别是，当你希望改变上下文环境之后并非立即执行，而是回调执行的时候，使用 bind() 方法。而 apply/call 则会立即执行函数。
